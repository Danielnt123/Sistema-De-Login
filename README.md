🔐 Sistema de Login

Este projeto é um sistema de login completo com suporte a diferentes tipos de usuários, incluindo:

👨‍🎓 Aluno

👨‍💼 Funcionário

🛠️ Administrador


O sistema também possui cadastro com envio de informações como e-mail e número de WhatsApp.

🚀 Funcionalidades

✅ Cadastro de usuários
✅ Login com autenticação
✅ Identificação de tipo de usuário
✅ Interface moderna com tema azul escuro
✅ Validação de dados
✅ Estrutura pronta para integração com backend

🖥️ Tecnologias utilizadas

HTML5

CSS3

JavaScript

(Pode ser adaptado para usar com backend como Node.js, PHP ou Firebase)

Página de Login

Este projeto é uma página de login simples desenvolvida utilizando HTML, CSS e JavaScript.
O objetivo é criar uma interface moderna e responsiva para autenticação de usuários.

Funcionalidades

- Interface de login moderna
- Campos de usuário e senha
- Validação básica de dados
- Design responsivo

Tecnologias utilizadas

- HTML
- CSS
- JavaScript

Objetivo do projeto

Este projeto foi criado para fins de aprendizado e prática em desenvolvimento web, demonstrando a criação de uma interface de autenticação simples.
